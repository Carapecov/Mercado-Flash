# Mercado Rápido Digital | Modelagem SQL

Este projeto foi desenvolvido em SQL para simulação de um banco de dados de um Mercado Rápido Digital, focado em integridade e segurança dos dados.

Adiciono este README como forma de documentar o aprendizado em SQL, a evolução de habilidades e o uso correto da tipagem de dados.

## Tecnologias Utilizadas

1. **SQL Puro:** Utilização de banco de dados com SQL padrão para organização de tabelas, tratamento rigoroso de tipos e prevenção de erros lógicos.
   * **Destaque:** Uso intensivo de *Constraints* (`CHECK`, `FOREIGN KEY`) para blindar o banco contra dados inválidos.

2. **Google Antigravity:** Nova IDE do Google baseada em IA (*Agent-First*). Utilizei-a como ferramenta principal para desenvolvimento, aproveitando suas ferramentas utilitárias e IA embutida para otimização e performance de código.

## Aprendizado

Este projeto ajudou a consolidar a importância da **Arquitetura de Dados**: evitar conflitos e planejar como cada tabela se comporta e conversa entre si. A visão estrutural do banco ajuda a projetar modelos escaláveis e evita que servidores tenham falhas de segurança na camada de dados.

* **Integridade e Atomicidade:** Foco em evitar "furos" no estoque ou dados órfãos utilizando travas e restrições nativas do próprio banco de dados.
